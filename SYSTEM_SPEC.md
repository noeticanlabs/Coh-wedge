## What This System Is

**A Coherence Engine for Computation**

It:
1. **Constructs** structure (Wedge 1-2)
2. **Normalizes** it (Wedge 3)
3. **Verifies** it (Wedge 4)
4. **Secures** it (Merkle hashing)

## Verification Pipeline

```

        │
        ▼
   verify_micro()
        │
        ▼
Chain of Receipts (linked sequence)
        │
        ▼
   verify_chain()
        │
        ▼
Slab (compressed macro-receipt)
        │
        ▼
   build_slab() → verify_slab() → verify_slab_with_leaves()
        │
        ▼

```

## Reject Codes (Failure Modes)

| Code | Condition |
|------|-----------|
| RejectSchema | Invalid schema_id or version |
| RejectCanonProfile | Canon profile hash mismatch |
| RejectChainDigest | Digest linkage failure |
| RejectStateHashLink | State transition discontinuity |
| RejectNumericParse | Invalid decimal string |
| RejectOverflow | Arithmetic overflow |
| RejectPolicyViolation | Accounting law inequality violated |
| RejectSlabSummary | Slab macro-accounting failure |
| RejectSlabMerkle | Slab Merkle root mismatch |
