# Coh Integrity Dashboard

Premium React dashboard for observing computational trust in Coh receipt chains. 

## Features
- **Authoritative Verification**: Backed by the Rust `coh-sidecar` (Port 3030).
- **Integrity Visualizer**: Tracks `TRUSTED` vs `TAMPERED` transitions.
- **Audit Inspector**: Peer into the "Accounting Law" math for any step.

## Setup
```bash
npm install
npm run dev
```
